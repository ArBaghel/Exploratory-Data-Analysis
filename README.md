Welcome to my repository for Exploratory Data Analysis (EDA) projects! Each notebook dives into real-world datasets using Python libraries like `pandas`, `numpy`, and `plotly`. Whether you're here for food trends in Bhopal or interactive visualizations of world population data, you're in the right place.
