# Exploratory Data Analysis Projects

Welcome to my repository for Exploratory Data Analysis (EDA) projects! Each notebook dives into real-world datasets using Python libraries like `pandas`, `numpy`, and `plotly`. Whether you're here for food trends in Bhopal or interactive visualizations of world population data, you're in the right place.

### 1. Bhopal Food Analysis
- Dataset: Local food preferences and vendor data from Bhopal.
- Objectives:
  - Explore popular dishes and cuisines.
  - Examine price distributions and review patterns.
  - Visualize ratings by location.
- Insights: Offers a glimpse into regional food habits and customer sentiment.

### 2. Loan Data EDA
- Dataset: Loan application and approval records.
- Objectives:
  - Identify patterns in loan approvals.
  - Analyze applicant demographics and credit history.
  - Detect correlations that influence loan outcomes.
- Insights: Supports data-driven lending decisions and risk modeling.

### 3. Netflix Titles Analysis
- Dataset: Netflix titles and metadata.
- Objectives:
  - Explore genre distribution and production trends.
  - Visualize country-wise content diversity.
  - Analyze title frequency over time.
- Insights: Highlights Netflix‚Äôs global storytelling evolution and content strategy.

### 4. World Population Analysis üåç
- Dataset: Global population data.
- Tools: `plotly.express`, choropleth maps.
- Objectives:
  - Interactive world map of population distribution.
  - Time-based trends in population growth.
  - Analyze by continent, income level, and development status.
- Insights: Helps visualize demographic changes and regional shifts.
